# Code of Conduct

We're committed to keeping Retrace a welcoming, low-drama place to
contribute. This document is short on purpose.

## Be decent

- Assume good intent in code review and issue threads.
- Critique the code, not the person.
- Help newcomers — everyone was new to Retrace once.

## Don't

- Harass, threaten, or attack anyone.
- Post discriminatory or sexual content.
- Disclose private information without consent.
- Use Retrace's GitHub footprint (issues, PRs, discussions) to promote
  unrelated commercial products.

## If something goes wrong

Open a private email to **conduct@retrace.dev** (or DM a maintainer
listed in [CONTRIBUTING.md](CONTRIBUTING.md) if email isn't an option).
We'll triage and respond within a week.

Repeated or severe violations result in being asked to leave the
project's spaces. We try to give one clear warning first; we won't
always have that luxury.

## Scope

This applies to GitHub issues, pull requests, discussions, the Retrace
Discord/Slack (if/when it exists), and any official Retrace channel.

## Attribution

This Code of Conduct is loosely inspired by the
[Contributor Covenant](https://www.contributor-covenant.org/) but is
intentionally short — we'd rather have you reading the code than the
governance docs.
